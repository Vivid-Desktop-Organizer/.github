# Vivid Desktop Organizer

**Vivid Desktop Organizer** is a Mac utility that brings color, style, and visual order to your file system. As a powerful **Vivid Folder Colorizer**, you can assign custom colors, gradients, and badge icons to any folder with a single click. This **Vivid Icon Manager** supports drag-and-drop icon replacement from custom collections or built-in presets. With **Vivid Workspace Customizer**, you can also change Finder sidebar colors, label presets, and folder backgrounds for a completely personalized organization system.

## Features

- **Folder Coloring**: The **Vivid Desktop Organizer** applies colors to folders without changing system files.
- **Badge System**: Use **Vivid Folder Colorizer** to add star, flag, heart, or emoji badges.
- **Icon Library**: As a **Vivid Icon Manager**, it includes 500+ premium icons for apps and folders.
- **Finder Tweaks**: This **Vivid Workspace Customizer** alters sidebar colors and grid spacing.
